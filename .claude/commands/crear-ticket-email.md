# Crear ticket desde email

Tomá el siguiente texto de email y creá un ticket en el EBS Support Portal.

Pasos:
1. Analizá el email para identificar: empresa, problema, prioridad sugerida
2. Generá un título conciso (no copies el asunto literalmente)
3. Escribí una descripción clara del problema
4. Usá el script: `node scripts/gmail-to-tickets.mjs` o creá el ticket manualmente via Supabase
